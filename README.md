# 🇮🇳 India Macro Indicator Dashboard (2015–2025)

## 📌 Overview

This project presents a comprehensive analysis of India’s macroeconomic performance from 2015 to 2025 using an interactive Power BI dashboard.

The objective is to track key economic indicators, identify trends, analyze relationships, and derive policy-relevant insights using data-driven methods.

---

## 📊 Indicators Covered

* **GDP Growth**
* **Inflation (CPI)**
* **Fiscal Deficit**
* **Trade & Current Account Deficit (CAD)**
* **Exchange Rate (INR/USD)**
* **Foreign Direct Investment (FDI)**

---

## 🔍 Analysis Methods

The dashboard applies multiple analytical approaches:

* **Time-Series Analysis** → Trend identification and economic cycles
* **Correlation Analysis** → Relationships between macro variables
* **Scenario Analysis** → Bull / Base / Bear economic outlook
* **Comparative Insights** → Cross-indicator interpretation

---

## 📈 Key Insights

* India experienced a **sharp GDP contraction in 2020**, followed by strong recovery
* Inflation spikes were linked to **global supply shocks and currency depreciation**
* Fiscal deficit widened during crisis periods and remains elevated
* Persistent **trade deficit drives CAD volatility**
* Rising FDI inflows indicate **long-term investor confidence**
